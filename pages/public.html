A New Demo for TheatreJS

16 Nov 2014

Hey there! The video you’re about to see is a web experiment done with TheatreJS.

Just a reminder that TheatreJS is an upcoming animation software for web designers and developers. It allows them to create all sorts of animation on the web, and deploy them as interactive web-based movies, ads, apps, and games.

TheatreJS is in active development. You can find our first published demo here.

In the demo you’ll see on this page, we’ve tried to experiment with syncing audio/video on a web browser. Every piece of this video is rendering live in your browser, so there are no <video> tags.

This is the credit roll of ‘The Dark Knight’ played to Hans Zimmer’s ‘Like a Dog Chasing Cars.’
